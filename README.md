## Intro to Backend

### Description
This project is a backend service developed with ***Express.js***. It follows the test-driven development(TDD) approach to building different services.
This web service will
- validate data, introduce internationalization, json manipulation, caching and static resource serving.
- handle interaction with external services
- handle database operations with ***Sequelize (ORM)***
- handle database versioning and migrations
